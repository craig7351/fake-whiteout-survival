# 雪地肉舖 Fake Whiteout Survival

一款雪地主題的**放置經營（idle tycoon）**小遊戲，靈感來自買量廣告常見的「雪地賣肉舖」。
技術棧與美術風格沿用姊妹專案 `winter`（Vue 3 + Vite + TypeScript + Tailwind + Babylon.js）。

## 玩法循環

1. **🐄 牧場**：店面後方圍出的牛圈，牛群持續遊蕩並在被獵殺後不斷重生；靠近時牛偶爾會衝過來頂你（會扣血、被擊退，停手後自動回血）。
2. **打牛獵肉**：走進牧場靠近牛即自動以裝備中的武器攻擊；牛被打死會爆出數塊肉掉在地上。
3. **撿肉搬運**：走過掉落的肉自動撿進背包，背在身上會疊高。
4. **🥩 販售攤位**：走回攤位把背上的肉擺上去陳列。
5. **顧客**：客人從前門排隊進場，買走攤位上的肉並付錢。
6. **💲 收銀**：客人付的錢堆在收銀格，走過去把錢收進錢包（金條飛回背上）。
7. **🔫 武器**：基地內有三個武器框，走上去站著付款買滿即解鎖並裝備；已購買的踩上去即可切換：
   - 🗡️ 大砍刀（起始免費，攻速快、可一次掃到多隻）、🪓 斧頭（單擊高傷）、🔫 衝鋒槍（遠距離快速掃射）

> 場地分兩區：**店面**（擺攤、收銀、換武器）與北側的**牧場**（獵牛），中間以柵欄缺口連通的走廊往返。

## 操作

- **行動**：左下角虛擬搖桿，或鍵盤 `WASD` / 方向鍵。
- **互動全自動**：走到攤位／收銀／武器框旁，站著不動即會自動進行（擺肉、收錢、購買或切換武器）。

## 開發

```bash
npm install
npm run dev      # 開發伺服器（http://localhost:5173）
npm run build    # 型別檢查 + production build
npm run preview  # 預覽 build 結果
```
